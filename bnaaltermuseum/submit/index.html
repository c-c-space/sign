This is a Internet Art Work Create Using Only 35 Colors and 18 Symbols.

You Can Join to Create This Work That Knows and Expresses Today's Feelings from Here.


これは、35 の 記号 と 18 の 色 を使って 自分の気持ちを知る・表す インターネットアート作品 です。

本日 BnA Alter Museume に 宿泊 する 皆様 は、この投稿フォームから 今の気持ち に合う 色 と 記号 を投稿し、この作品の制作に参加することができます。


この作品について

ページ背景のグラデーションカラーは、投稿された色が上から下へと連なり生成された、本日の宿泊者の皆様の気持ちを集めた色です。
※ グラデーションが流れる速度は、投稿数に比例し加速します。

ページ下部の日付をクリックすると、投稿された色と記号がフルディスプレイサイズで一つずつランダムに表示されます。
記号の色は、投稿された色の反対色で表示され、自分の気持ちの反対を知ることに作用します。
※ ページ上部のメーターを左右に操作すると、色と記号が入れ替わる速度／電子音が変化する速度が変化します。

ページ下部の現在時刻をクリックすると、投稿された全ての色と記号の記録が表示されます。
※ 投稿記録は、日本時間24:00に、毎日リセットされます。
