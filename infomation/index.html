This is a Communication Tool using Colors and Symbols That Suits Our Feelings.

これは、35 の 記号 と 18 の 色 から 今の気持ちに合う 記号と色を投稿し、自分たちの気持ちを知る・表す オンラインコミュニケーションツール です。

概要

言葉を使うよりも手軽に簡単に、上手／下手などの優劣が付きづらい方法で、世界中の人達が平等に参加できるコミュニケーションの一つの形としてこのコンテンツを制作しました。

色と記号を使って、ここに自分の気持ちを投稿することや、集まったいろいろな人達の気持ちを眺めることが、皆様にとってのささやかな安心に繋がれば幸いです。

仕様

投稿された色は、上から下へと流れるカラーグラデーションに追加され、みんなの気持ちをまとめたカレンダーのような表示の一部となります。
(新しい投稿は、カラーグラデーションの一番下に追加されます。)

Flashをクリックすると、投稿された色と記号がフルディスプレイサイズで一つずつランダムに表示されます。
画面中央に表示される記号は自分の気持ちを表す色の反対色となる仕様となっており、自分の気持ちの反対を知ることに作用します。
（ページ上部のメーターを操作すると、表示が入れ替わる速度が変化します。）

Only Members Can be Posting Color and Symbol Here and Can Browsing All of The Collection.

※ このコンテンツの平和を守るため、色と記号の投稿はコミュニティ会員のみに制限しています。

コミュニティ会員への入会は無料です。
投稿への参加を希望される方は、お気軽にご入会ください。
